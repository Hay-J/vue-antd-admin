# vue-antd-admin
## 1. 将仓库拉到本地或者下载到本地
```
git clone https://github.com/Hay-J/vue-antd-admin.git
```

## 2. 进入项目目录安装依赖
```
npm install（被墙可以用cnpm或yarn）
```

## 3. 运行开发时环境
```
npm run dev
```
## 4. 浏览器'http://localhost:8080/'预览
